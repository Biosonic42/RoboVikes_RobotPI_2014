RoboVikes_RobotPI_2014
======================

Code for Android Scouting Application for 2014

Welcome to RobotPI, the fully comprehensive scouting system for FRC season 2014.  This application includes scouting through android tablets to record individual robot performance, known as Function FIRST, as well as a complete Statistical Analysis Database (SAD) to interpret data collected.

FIRST Team 701 the RoboVikes designed and uses RobotPI to scout during their FRC Season.  Updates for the new game coem around the end of that years build season.
